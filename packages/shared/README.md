# @evm-effect/shared

Shared utilities for the evm-effect monorepo.

## Contents

- **Test Utilities** (`@evm-effect/shared/test`): Python comparison testing, fast-check arbitraries
- **Common Helpers**: Shared functions used across packages

## Usage

```typescript
// Import test utilities
import { pythonEval, testAgainstPython } from "@evm-effect/shared/test";
```
